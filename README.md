# RCWL-0516
Allt om RCWL-0516 och demoprojekt

| PIN | Notes                     |
|-----|---------------------------|
| 3V3 | 3V3 Control power out     |
| GND | Ground                    |
| OUT | when detect, output HIGH  |
| VIN | Input voltage 4-28V       |
| CDS | Enable control chip (<0V7 = OUT LOW) |      |
